API: Application Programming Interface

    API :
        1)Request
          1)Protocol: HTTP/HTTPs
          2) URI
            URI = Uniform Resource Identifier
            URI = URL + URN
            URI = Uniform Resource Locator + Uniform Resouce Name
            Example:
            https://www.mr-dev.tech/mern
            URI: https://www.mr-dev.tech/mern
            URI = URL + URN
            URL = https://www.mr-dev.tech
            URN = mern
            SERVER IP: 45.87.81.78
            PING IP: 104.21.6.156
          3) HTTP Request Method
            GET: Retrieve Data from data source
            POST: Save or Process data
            PUT/PATCH:
              PUT: Send data for update or create if not existed
              PATCH: Send data for update and nothing happen if not existed.
            DELETE: Delete data from datasoure
          4) Request Data:
          POST,PUT,PATCH
            Data can be sent to server side using Request BODY
            Request Body Types:
              =>KEY: Value
                Key always String
                Value: Can be from any type, converted to string
              => form-data
                Used to send
                  Files
                  Text
              => x-www-form-url-encoded
                Used to send
                Text
          5)Headers
            Consists of Key & Value
            Keys & Values are string
            Examples:
              Has an expiration date
              Unique for each user
              Can't be duplicated
              has many types (Bearer)
        2) Response
          => Body
            Body
            Types
              JSON
                Java Script Object Notation
                Each object Consists of KEY: Value
                  - Key is always "String"
                  - Must add : Between key and value
                  - At the end of each key: value pairs we must add , if there is a new pairs after!
                  - Value types:
                    *String
                    *numeric
                    *bool
                    *null
                    *object
                    *array
                JSON Object {}
                JSON Array []
              XML
          => Response Status Code
            Consists of Textual Value and number
            Examples:
              2xx
                200 OK
                201 Created
                202 Accepted
                204 No Content
              4xx
                400 BadRequest
                401 UnAuthorized
                403 Forbidden
                404 NotFound
                405 Method Not Allowed
              5xx
                500 ServerError
          => Execution Time: (ms/s)
          => Response Size: (b,kb,m)
